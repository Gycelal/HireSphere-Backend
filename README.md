# HireSphere Backend

Backend for **HireSphere**, built with **Python + Django/FastAPI**.  
Handles authentication, role-based access, recruiter approval, and API endpoints for frontend consumption.

## Features

- Role-based authentication (Candidate, Recruiter, Admin)
- Google OAuth login
- Admin dashboard APIs for recruiter approval
- Candidate and recruiter management
- Token-based authentication (JWT or session-based)
- API endpoints for frontend integration

## Getting Started

### Prerequisites

- Python 3.10+  
- pip or Poetry  
- PostgreSQL / MySQL / SQLite (depending on your setup)
